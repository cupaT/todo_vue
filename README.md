# To-Do List (Vue 3 + Vite)


## Деплой

`cupat.me/todo_vue`


## Описание

Проект — простой To‑Do List с возможностью переключения темы, поиска, редактирования и удаления задач.  
Текущая реализация выполнена на **Vue 3 (Composition API)** с использованием **Vite** для сборки и **Sass/SCSS** для оформления.

Приложение поддерживает сохранение задач и выбранной темы в `localStorage`, поэтому состояние не теряется при перезагрузке страницы.

## Функционал

- Список задач (to‑do items)
- Карточка задачи содержит:
  - чекбокс выполнения
  - название задачи
  - кнопки редактирования и удаления
- Добавление задачи через модальное окно
- Редактирование задачи (смена названия)
- Удаление задачи
- Переключение темы (светлая / тёмная)
- Поиск по названию (динамическая фильтрация)
- Выполненные задачи перемещаются в отдельный список и отображаются зачёркнутыми
- Адаптивный дизайн для мобильных и десктопных устройств
- Сохранение задач и темы в `localStorage`

## Технологии

- [Vue 3](https://vuejs.org/) (Composition API)
- [Vite](https://vitejs.dev/) — сборка и запуск проекта
- Sass/SCSS — стилизация интерфейса
- Tailwind CSS — **не используется** (вёрстка на собственных SCSS‑модулях)
- LocalStorage — хранение задач и выбранной темы

## Структура проекта

Основные файлы:

- `index.html` — точка входа Vite (контейнер для Vue‑приложения)
- `src/main.js` — инициализация Vue‑приложения
- `src/App.vue` — корневой компонент, хранит состояние и управляет логикой
- `src/components/`
  - `TodoItem.vue` — компонент отдельной задачи
  - `TodoList.vue` — список задач
  - `AddTodoModal.vue` — модальное окно добавления/редактирования задачи
  - `SearchBar.vue` — поиск по задачам
  - `ThemeToggle.vue` — переключение темы
- `src/styles/main.scss` — точка входа для стилей
- `src/styles/scss/*.scss` — модули стилей (layout, header, todo‑карточки, модалки, темы и т.п.)
